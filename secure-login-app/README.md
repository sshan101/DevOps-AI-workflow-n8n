# Secure Login Application

Enterprise Java web application with modern, animated login functionality following best practices.

## Features
- Modern, responsive login UI with animations
- BCrypt password hashing
- Security headers and CSRF protection
- Session management
- Input validation (client and server-side)
- Clean MVC architecture
- JUnit test coverage

## Tech Stack
- Java 11
- Maven
- Servlet 4.0
- JSP/JSTL
- BCrypt for password hashing
- SLF4J + Logback for logging
- JUnit 5 + Mockito for testing

## Build Instructions

mvn clean package
